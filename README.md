# hibernate-1
```
准备:在Eclipse MarketPlace下载工具Jboss tools
1.快速创建一个Hibernate实例
     1.导入jar包
     2.使用工具创建Hibernate配置文件hibernate.cfg.xml
     3.创建与数据库里面的表对应的实体类
     4.使用工具创建对象关系映射文件
     5.通过Hibernate api编写访问数据库代码
        1.创建SessionFactory,并将配置或者服务注册到服务注册类中
        2.获取Session
        3.开启事务管理
        4.执行数据库操作
        5.提交事务(不然不执行)
        6.关闭session
        7.关闭factory```
